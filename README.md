# 🛡️ Password Security Analyzer

Bu layihə istifadəçilərin şifrə təhlükəsizliyini analiz etmək üçün hazırlanmış interaktiv bir veb-tətbiqdir.

## ✨ Funksiyalar
- **Live Analysis:** Şifrənin gücünü anında yoxlayır.
- **Brute-Force Prediction:** Şifrənin qırılma müddətini riyazi olaraq hesablayır.
- **Glassmorphism UI:** Modern və şəffaf istifadəçi interfeysi.
- **Security First:** Bütün hesablamalar lokal olaraq brauzerdə aparılır.
